## ğŸ§  KEY PRINCIPLES BUILT IN

### The Neuroscience Foundation
- **Reptilian Brain:** Safety first (pattern interrupt, permission, warm tone)
- **Limbic Brain:** Emotion drives decisions (pain amplification, vision creation)
- **Neocortex:** Logic justifies (specific data, ROI, case studies)

### Jim Camp's "Start with NO"
- **Not Needy:** Mission over money, serve don't sell
- **Respect Autonomy:** Give permission to say no throughout
- **Blank Slate:** No assumptions, genuine curiosity
- **Focus on Their World:** 80% questions about them
- **Decisions, Not Emergencies:** Exploration, not pressure

### Human Connection Over Tactics
- **Listen 70% / Talk 30%:** Understanding > convincing
- **Validate Before Reframe:** Respect concerns, don't argue
- **Disqualify Bad Fits:** Good NO > bad YES
- **Authentic Voice:** Sound human, not scripted
- **Serve First:** If you can't help them, say so
